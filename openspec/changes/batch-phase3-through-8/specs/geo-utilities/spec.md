# geo-utilities

## Requirements

- REQ-GEO-001: Haversine distance calculation between two GPS points
- REQ-GEO-002: Bearing calculation from point A to point B
- REQ-GEO-003: Destination point given origin, bearing, distance
- REQ-CONV-001: CoordConverter shall convert NED ↔ WGS-84 using reference point
- REQ-CONV-002: Conversion accuracy verified with known-answer tests
- REQ-UNITS-001: Degree/radian conversion helpers
- REQ-UNITS-002: m/s ↔ km/h conversion helpers
- REQ-UNITS-003: Meter ↔ foot conversion helpers
- REQ-TIMING-001: Precise timing helpers for performance measurement
- REQ-TIMING-002: Rate limiter for periodic operations
